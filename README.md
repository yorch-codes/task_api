
# Project Name

A simple API Tasks created whit FastAPI, use the local memory to store tasks.

---

## Features

* Create tasks
* Read tasks
* Update tasks
* Delete tasks

---

## Tech Stack

* Python 3
* FastAPI

---

## Installation

Clone the repository:

```bash
git clone git@github.com:yorch-codes/task_api.git
cd task_api
```

Create environment and install dependencies:

```bash
uv sync
```

---

## Usage

Run the project:

```bash
python main.py
```
